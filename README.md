# Push-up Counter (v2)

## Run
python src/main.py

## Streamlit 

streamlit run src/streamlit_app.py

## Controls
- q: quit
- r: reset current set count
- s: toggle debug
- n/p: decrease/increase target reps per set

## What gets saved
- JSON + CSV per session in data/sessions/
- Fields: date/time, total reps, number of sets, reps per set, duration, avg time per rep, parameters used.
